## A complete MERN based Notes app (in-progress)

### Techs Used:
1. React.js (using Vite)
2. Tailwind CSS
3. Express.js
4. MongoDB (mongoose)

### How to run the app:
On your terminal, run the following commands:
1. `git clone https://github.com/starDust1703/Notes.git`
2. `cd Notes`
3. `npm install`
4. `npm run dev`
5. On your browser, open http://localhost:3000

## App Workflow

### /backend
server.js -> express routers -> middlewares -> controllers -> database

### /frontend
Auth -> signup -> login -> home -> addNote
