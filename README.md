## A complete MERN based Notes app (in-progress)

### Techs Used:
1. React.js (using Vite)
2. Tailwind CSS
3. Express.js
4. MongoDB (mongoose)

### How to run the app:
On your terminal, run the following commands:
1. `git clone https://github.com/starDust1703/Notes.git`
2. `cd Notes\frontend`
3. `npm install`
4. `cd..`
5. `cd backend`
6. `npm install`
7. `npm run dev`
8. On your browser, open http://localhost:8080

## App Workflow

### /backend
index.js -> express routers -> middlewares -> controllers -> database

### /frontend
Auth -> (signup\login) -> home -> addNote
